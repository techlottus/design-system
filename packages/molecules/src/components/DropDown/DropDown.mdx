# DropDown 


The dropdown menu serves as a navigational aid, facilitating various actions such as accessing the menu, 
navigating to the profile, or logging out. It enables users to seamlessly transition to different sections
or paths within the same page.
Within the @lottuseducation/molecules library, the DropDown component comprises the following elements : 
"DropDown.Button","DropDown.Items" and "DropDown.Item". By leveraging these components, a multitude of 
distinct designs for DropDown can be crafted.

- DropDown: The primary component that encompasses all elements within the DropDown .
- DropDown.Button: Contains the children for the actionable button of the DropDown .
- DropDown.Items: Container for the options side of the DropDown. 
- DropDown.Item: Each option of the Items container. 

## Installation and import

To integrate this component, we leverage the @lottuseducation/molecules library. This library seamlessly combines unstyled
yet fully accessible UI components into our projects, meticulously designed to complement Tailwind CSS integration.

To get started, initiate the installation of @lottuseducation/molecules via npm:

```
   npm install @lottuseducation/molecules
```


After installing the library, you can import the component into your project as follows:

`import DropDown from "@lottuseducation/molecules"`


## Usage

To incorporate the DropDown component into your code, follow the structure demonstrated below:

```
 <DropDown >
    <DropDown.Button>button content</DropDown.Button>
    <DropDown.Items>
    <DropDown.Item>content item 1</DropDown.Item>
    <DropDown.Item>content item 2</DropDown.Item>
    </DropDown.Items>    
 </DropDown>
```
The previous example showcases the simplest usage of the DropDowncomponent, utilizing default props and rendering an DropDown
as illustrated below:
{
    <DropDown >
    <DropDown.Button>button content</DropDown.Button>
    <DropDown.Items>
    <DropDown.Item>content item 1</DropDown.Item>
    <DropDown.Item>content item 2</DropDown.Item>
    </DropDown.Items>    
    </DropDown>
}

You can easily customize by incorporating Tailwind CSS classes. Take, for instance, the following example, where we've applied shadow effects to both the items and the dropdown button, while also establishing a fixed height for the items container.
```
<DropDown className="shadow">
    <DropDown.Button>Button</DropDown.Button>
    <DropDown.Items className="h-40">
      <DropDown.Item >option</DropDown.Item>
      <DropDown.Item >option</DropDown.Item>
      <DropDown.Item >option</DropDown.Item>
      <DropDown.Item >option</DropDown.Item>
    </DropDown.Items>    
</DropDown>
```
