# Select

The Select component is a user interface element that enables users to choose a predefined option from a drop-down list,
commonly used in forms for selecting options.

This component includes a selection box and a drop-down list containing available options. When a user clicks the selection box,
the list of options appears, allowing the user to make a choice by clicking on an option. Once an option is selected, 
the list closes, and the chosen option is displayed in the selection box.

Within the @lottuseducation/molecules library, the select  component comprises the following elements : "Select.Button,"
   "Select.Options" and "Select.Option". By leveraging these components, a multitude of distinct designs for select  can be crafted.
   
- Select: The primary component that encompasses all elements within the select .
- Select.Button: The selection box .
- Select.Options: The dropdown list container
- Select.Option: Each option in dropdown list container


## Installation and import

To integrate this component, we leverage the @lottuseducation/molecules library. This library seamlessly combines unstyled
yet fully accessible UI components into our projects, meticulously designed to complement Tailwind CSS integration.

To get started, initiate the installation of @lottuseducation/molecules via npm:

```
   npm install @lottuseducation/molecules
```


After installing the library, you can import the component into your project as follows:

`import Select from "@lottuseducation/molecules"`

## Using states to change value

To add functionality in select, follow the example below:

```
const Ejemplo = (args) => {
  const people = [
    { id: 1, name: 'Durward Reynolds', unavailable: false },
    { id: 2, name: 'Kenton Towne', unavailable: false },
    { id: 3, name: 'Therese Wunsch', unavailable: false },
    { id: 4, name: 'Benedict Kessler', unavailable: true },
    { id: 5, name: 'Katelyn Rohan', unavailable: false },
  ]
  
    const [selectedPerson, setSelectedPerson] = useState({name:"Name"})


    return (
      <div className="w-52">
      <Select value={selectedPerson} 
      onChange={setSelectedPerson}  >
        <Select.Button {...args} ><span className="w-4 font-icons-solid text-surface-400 mr-2">people</span> {selectedPerson.name}</Select.Button>
        <Select.Options  className={cn("h-40")}>
          {people.map((person) => (
            <Select.Option
              key={person.id}
              value={person}
              disabled={!args.disabled?person.unavailable : args.disabled}
            >
              {person.name}
            </Select.Option>
          ))}
        </Select.Options>
      </Select></div>
    )
  }


```
