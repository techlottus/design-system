# Iconography
<Callout emoji="ðŸ’¡">
  If you want to use `@bedu-tmp/iconography` you must have a latest version from `react`
</Callout>

## Installation
```sh
yarn add @bedu-tmp/iconography
```

## Quick start
```jsx
import { ArrowIcon } from "@bedu-tmp/iconography"

const SinglePage = () => {
  return (
    <div>
      <ArrowIcon className="w-5 h-5" />
    </div>
  );
}
```
