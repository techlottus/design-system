# Input

Documentation for the `<Input />` component. Learn about the available props and how to use it.

## Installation

To get started, install `@exponentialeducation/betomic` via `yarn`:

```bash
# Yarn

yarn add @exponentialeducation/betomic
```

## When to use

A user input is needed.

```jsx highlight=1,8
import { Input } from '@exponentialeducation/betomic';

function IndexPage() {
  return (
    <main>
      {/* ... */}

      <Input placeholder="Placeholder" />

      {/* ... */}
    </main>
  )
}
```
