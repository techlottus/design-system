# Container

Documentation for the `<Container />` component. Learn about the available props and how to use it.

## Installation

To get started, install `@bedu-tmp/betomic` via `yarn`:

```bash
# Yarn

yarn add @bedu-tmp/betomic
```

## Basic Example

A `Container` are used to pad the content inside of them.

```jsx highlight=3,10,16
 {/* ... */}

import { Container } from '@bedu-tmp/betomic';

function IndexPage() {
  return (
    <main>
      {/* ... */}

      <Container>
        <p>...</p>
        <p>...</p>
        <p>...</p>
        <p>...</p>
        <p>...</p>
      </Container>

      {/* ... */}
    </main>
  )
}
```

## Fluid

Five sizes are available.

```jsx highlight=3,10,16
 {/* ... */}

import { Container } from '@bedu-tmp/betomic';

function IndexPage() {
  return (
    <main>
      {/* ... */}

      <Container fluid>
        <p>...</p>
        <p>...</p>
        <p>...</p>
        <p>...</p>
        <p>...</p>
      </Container>

      {/* ... */}
    </main>
  )
}
```
