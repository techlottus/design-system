# Avatar

Documentation for the `<Avatar />` component. Learn about the available props and how to use it.

## Installation

To get started, install `@bedu-tmp/betomic` via `yarn`:

```bash
# Yarn

yarn add @bedu-tmp/betomic
```

## Basic Example

An `Avatar` can be used to represent people or objects.

```jsx highlight=3,10-14
 {/* ... */}

import { Avatar } from '@bedu-tmp/betomic';

function IndexPage() {
  return (
    <main>
      {/* ... */}

      <Avatar>
        <img
          src="https://picturepan2.github.io/spectre/img/avatar-1.png"
          alt="Imaginary Friend"/>
      </Avatar>

      {/* ... */}
    </main>
  )
}
```
<div className="mt-5 flex justify-center">
  <Avatar>
    <img
      src="https://picturepan2.github.io/spectre/img/avatar-1.png"
      alt="Imaginary Friend"/>
  </Avatar>
</div>

## Sizes

Five sizes are available.

```jsx highlight=3,10
 {/* ... */}

import { Avatar } from '@bedu-tmp/betomic';

function IndexPage() {
  return (
    <main>

      <Avatar size="xs">
        <img
          src="https://picturepan2.github.io/spectre/img/avatar-1.png"
          alt="Imaginary Friend"/>
      </Avatar>

      {/* ... */}
    </main>
  )
}
```
