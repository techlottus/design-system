# Input Group

Documentation for the `<InputGroup />` component. Learn about the available props and how to use it.

## Installation

To get started, install `@exponentialeducation/betomic` via `yarn`:

```bash
# Yarn

yarn add @exponentialeducation/betomic
```

## Basic Example

`InputGroup` are built using the `InputGroup.Decoration` component.

The `InputGroup.Decoration` place the decoration element in `right` or `left` position,
by default the elements are placed in the left side.

`InputGroup` are a basic building block used to render text inputs across many components.
They allow you to add icons and buttons within a text input to expand its appearance and functionality.
For example, you might use an input group to build a visibility toggle for a password field.

```jsx highlight=3,10-15
 {/* ... */}

import { Input, InputGroup } from '@exponentialeducation/betomic';

function IndexPage() {
  return (
    <main>
      {/* ... */}

      <InputGroup>
        <InputGroup.Decoration position="right">
          <EyeIcon className="w-4 h-full" />
        </InputGroup.Decoration>
        <Input type="password" className="pr-10" placeholder="Right decoration" />
      </InputGroup>

      {/* ... */}
    </main>
  )
}
```
