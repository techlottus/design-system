# SelectChip

The Select Chip component enables users to choose from a list of options, selecting the one they prefer. It functions as a type of label that requires multiple Select Chips to be effective and meaningful.

These components can be used both within forms and independently, whenever users need to select an item from a list of options.

## Installation

To begin, start the installation process of Headless UI using npm.

`npm install @lottuseducation/molecules`

## Import

You can utilize the SelectChip component by importing it as follows:

`import SelectChip from "../components/SelectChip"`

Here are one examples showcasing the use cases for the SelectChip and its functionality:

## SelectChip

SelectChips are constructed using the RadioGroup and RadioGroup.Option components.

Select an option by clicking on it, and when the SelectChip is focused, click again to uncheck the option.

 <SelectChip
            iconActive={<RadioButtonActive />}
            iconInactive={<RadioButtonDefault />}
            className="border-success-500 w-24"
            active={true}
            label="Label"
        />

## SelectChip Group

SelectChips are constructed using the RadioGroup and RadioGroup.Option components.

Select an option by clicking on it, and when the SelectChip is focused, click again to uncheck the option.

<RadioGroup className="grid gap-3 w-1">
    <RadioGroup.Option value="">
        <SelectChip
            iconActive={<RadioButtonActive />}
            iconInactive={<RadioButtonDefault />}
            className="border-success-500"
            active={true}
