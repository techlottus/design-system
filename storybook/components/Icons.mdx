# Lottus Education Icons

@lottusicons features the official Lottus Icons meticulously converted into SvgIcon components for seamless integration.

Every icon is encapsulated within its own component, derived directly from its SVG file. These components offer the flexibility to apply various attributes, allowing for customized styling.

Icons come with default styles. The simplest method to utilize an icon is as follows:

```
import * from "@lottuseducation/icon"

export default function MyApp() {

return (
        <CheckBoxActive />
    );
};

```

If you utilize the icon without specifying attributes, you will receive a standard 16-pixel icon in black color by default.

<CheckBoxActive size="sm" />

# Attributes

The attributes icons can receive are as follows:
