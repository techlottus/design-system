# Input 

The Input  is a fundamental component commonly employed within forms,
 facilitating user input across various data types such as text, numbers, emails, and passwords, among others.
  It serves as a conduit for both sending and collecting inputted data. Within the @lottuseducation/molecules library, 
  the Input  component comprises two optionals elements : "Input.RightElement," and
   "Input.LeftElement." By leveraging these components, a multitude of distinct designs for input  can be crafted.
   
- Input: The primary component that encompasses all elements within the input .
- Input.RightElement: Contains the children for the right section of the input .
- Input.LeftElement: Holds the children for the left side of the input 

## Installation and import


To integrate this component, we leverage the @lottuseducation/molecules library. This library seamlessly combines unstyled yet fully accessible UI components into our projects, meticulously designed to complement Tailwind CSS integration.

To get started, initiate the installation of @lottuseducation/molecules via npm:

```
   npm install @lottuseducation/molecules

```


After installing the library, you can import the component into your project as follows:

`import Input from "@lottuseducation/molecules`
 

## Usage


To incorporate the Input  component into your code, follow the structure demonstrated below:

```
<Input label="label" type="text" >

    //optional
    <Input.RightElement>
    ...childrens
    </Input.RightElement>

    //optional
    <Input.LeftElement>
    ...childrens
    </Input.LeftElement>

</ Input>

```
The previous example showcases the simplest usage of the Input component, utilizing default props and rendering an Input as illustrated below:
{
      <Input label = "label" type = "text" id="neutral-input">
         <Input.RightElement className="bg-primary-100"> <div className='w-3 h-4'></div> </Input.RightElement> 
         <Input.LeftElement className="bg-primary-100"><div className='w-3 h-4'></div> </Input.LeftElement>
      </Input>
               }
