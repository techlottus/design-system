# CheckBox

Checkboxes empower users to effortlessly select one or multiple items from a given set, enhancing their control and flexibility within the interface.

## Installation

To implement this component we use Headless UI which allows us to seamlessly combine unstyled but fully accessible UI components in our projects, 
meticulously designed to harmonize with the Tailwind CSS integration.

To kick off, initiate the installation of Headless UI via npm.

`npm install @headlessui/react`

## CheckBox

Checkboxes serve to activate or deactivate an option. When presenting multiple choices within a list, employing checkboxes rather than on/off switches conserves space. 
Conversely, for singular options, opting for an on/off switch over a checkbox is preferable.
