# CheckBox

Check boxes provide users with a seamless way to select one or more items from a designated set, enhancing their ability to control and customize their experience within the interface.

## Installation

To begin, start the installation process of Headless UI using npm. EN VEZ DE To kick off, initiate the installation of Headless UI via npm

`npm install @headlessui/react`

## Usage

Checkboxes are used to enable or disable an option. When presenting multiple choices in a list, using checkboxes instead of on/off switches helps conserve space.

To implement a checkbox in a list, refer to the following code example:

```
let [optionOne, setOptionOne] = useState(false)

<RadioGroup>
    <RadioGroup.Option value={optionOne} className="!w-5 !mx-2" onClick={() => setOptionOne(!optionOne)}>
        <>
            {optionOne ?
                <CheckBoxActive size="md" className="!mx-2 cursor-pointer" />
                : <CheckBoxDefault size="md" className="!mx-2 cursor-pointer" />
            }
        </>
    </RadioGroup.Option>       
</RadioGroup>
```

## CheckBoxLabel

You can utilize a checkbox list with labels by following the code example below:

```
let [optionSeven, setOptionSeven] = useState(false)

<RadioGroup>
    <RadioGroup.Option value={optionSeven} className="flex !mt-4 cursor-pointer !mb-3 w-20" onClick={() => setOptionSeven(!optionSeven)}>
        <>
            {optionSeven ?
                <CheckBoxActive size="md" className="!mx-2 cursor-pointer" />
                : <CheckBoxDefault size="md" className="!mx-2 cursor-pointer" />
            }
            <span className="">Label</span>
        </>
    </RadioGroup.Option>                                      
</RadioGroup>
```

## CheckBoxLink

To include a privacy link with a checkbox, refer to the following code example:
```
let [optionThirteen, setOptionThirteen] = useState(true)

<RadioGroup>
    <RadioGroup.Option value={optionThirteen} className="!flex mb-2 cursor-pointer mt-2" onClick={() => setOptionThirteen(!optionThirteen)}>
        <>
            {optionThirteen ?
                <CheckBoxActive size="md" className="!mx-2 my-auto" />
                : <CheckBoxDefault size="md" className="!mx-2 my-auto" />
            }
            <p className="">I have read the <a className="!underline" href="#"> Privacy Notice</a></p>
        </>
    </RadioGroup.Option>
</RadioGroup>
```
