# RadioGroup

RadioGroup presents a collection of components designed to solve the requirement of selecting an option from multiple options.

## Installation

To implement this component we use Headless UI which allows us to seamlessly combine unstyled but fully accessible UI components in our projects, meticulously designed to harmonize with the Tailwind CSS integration.

To kick off, initiate the installation of Headless UI via npm.

`npm install @headlessui/react`

Here are two examples showcasing the use cases for the radio group and its functionality:

## SelectChip

Radio Groups are constructed using the RadioGroup, RadioGroup.Label, and RadioGroup.Option components.

Select an option by clicking on it, and when the radio group is focused, utilize the arrow keys to navigate and change the selected option.

 <SelectChip /> 
```
import { useState } from 'react'
import { RadioGroup } from '@headlessui/react'
import React from "react";
import Icon from '../components/Icon/Icon';
const MyRadioGroup = () => {
 let [plan, setPlan] = useState('startup')

 return (
  <div className="w-full px-1 grid grid-cols-12">
   <div className="col-span-5">
    <RadioGroup value={plan} onChange={setPlan}>
     <RadioGroup.Option value="startup" className="bg-surface-100 w-24 h-9	border-2 border-black rounded py-1.5">
      {({ checked }) => (
       <>
        {checked ?
         <Icon iconName="adjust" className="text-base ms-2 me-3 text-dark" />
         : <Icon iconName="radio_button_unchecked" className="text-base ms-2 me-3 text-surface-400" />
        }
        <span className={checked ? 'bg-surface-100' : ''}>Label</span>
       </>
      )}
     </RadioGroup.Option>
     <RadioGroup.Option value="business" className="bg-surface-100 w-24 h-9 border-2 border-info-700 rounded py-1.5 my-3">
      {({ checked }) => (
       <>
        {checked ?
         <Icon iconName="adjust" className="text-base ms-2 me-3 text-info-700" />
         : <Icon iconName="radio_button_unchecked" className="text-base ms-2 me-3 text-surface-400" />
        }
        <span className={checked ? 'bg-surface-100' : ''}>Label</span>
       </>
      )}
     </RadioGroup.Option>
     <RadioGroup.Option value="enterprise" className="bg-surface-100 w-24 h-9 border-2 border-error-600 rounded py-1.5">
      {({ checked }) => (
       <>
        {checked ?
         <Icon iconName="adjust" className="text-base ms-2 me-3 text-error-600" />
         : <Icon iconName="radio_button_unchecked" className="text-base ms-2 me-3 text-surface-400" />
        }
        <span className={checked ? 'bg-surface-100' : ''}>Label</span>
       </>
      )}
     </RadioGroup.Option>
    </RadioGroup>
   </div>
  </div>
 )
};
```

You have the option to customize the shape of the items you wish to select; in this scenario, the options can be displayed as cards. 

You can utilize this pre-customized component and further tailor it to your preferences, or leave the current state as desired.
